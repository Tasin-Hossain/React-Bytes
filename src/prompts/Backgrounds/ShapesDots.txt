      { stops: ["#7a5af8", "#22c39b"] },

    ]}

    opacity={1}
    animationMode="off"

    {/* optional children render above canvas */}

### Props
| Prop | Type | Default | Description |
|------|------|---------|-------------|
| width | string | "100%" | CSS width of the outer wrapper div. |
| height | string | "100%" | CSS height of the outer wrapper div. |
| className | string | "" | Additional CSS classes applied to the outer wrapper div. |
| children | node | — | Optional child elements rendered above the canvas (no z-index wrapper — relies on normal DOM stacking order). |
| cellSize | number | 40 | Size in pixels of each grid cell. Controls both spacing between shapes and (combined with `overlapGuard`) the maximum shape size. Triggers a grid rebuild when changed. |
| influenceRadiusVmin | number | 30 | Radius of the cursor's influence zone, as a percentage of `vmin` (the smaller of canvas width/height). Shapes within this radius scale up toward their `peak` based on proximity. |
| attackTime | number | 0.5 | Time constant (seconds) for how quickly a shape grows toward its target scale. Smaller = snappier growth. |
| releaseTime | number | 0.6 | Time constant (seconds) for how quickly a shape shrinks back toward `idleScale` once no longer influenced. Smaller = snappier shrink. |
| idleScale | number | 0.09 | Resting scale multiplier for shapes with no cursor/burst/animation influence. |
| minPeakScale | number | 1 | Lower bound of each shape's randomly assigned "peak" scale (reached at full influence). Triggers a grid rebuild when changed. |
| maxPeakScale | number | 3 | Upper bound of each shape's randomly assigned peak scale. Each shape's peak is clamped by `overlapGuard` so neighbors can't visually collide even at max influence. Triggers a grid rebuild when changed. |
| burstSpeed | number | 1200 | Speed (px/sec) at which a click-triggered burst ring expands outward from the click point. |
| burstThickness | number | 180 | Thickness in pixels of the expanding burst ring. Shapes within this ring's width scale up as it passes through them. |
| backgroundColor | string | "#080808" | Canvas background fill. Use "transparent" to skip filling. |
| shapes | string[] | ["circle", "triangle", "square"] | Pool of shape types each grid cell randomly draws from. Triggers a grid rebuild when changed. |
| colors | array | see usage example | Pool of colors each grid cell randomly draws from. Each entry is either a hex string (solid fill) or an object `{ stops: [colorA, colorB] }` (vertical linear gradient fill). Triggers a grid rebuild when changed. |
| dpr | number | 2 | Device pixel ratio cap for canvas resolution (e.g. 1, 1.5, or 2). |
| opacity | number | 1 | Global alpha applied to all shapes while drawing (0–1). |
| animationMode | string | "off" | Ambient animation pattern applied independently of cursor/click interaction. See the **Animation Modes** table below. |
| animationSpeed | number | 1 | Multiplier on the internal animation clock driving `animationMode` patterns. |
| overlapGuard | number | 0.86 | Fraction (0–1) of half a cell's size that a shape is allowed to reach at its peak. Lower values leave more breathing room between neighboring shapes; `1.0` allows shapes to nearly touch. Triggers a grid rebuild when changed. |
| rotationVariance | number | 1 | Multiplier (0–1) on each shape's randomly assigned static rotation. `0` = no shapes rotated; `1` = full random rotation range. Triggers a grid rebuild when changed. |

### Animation Modes
The `animationMode` prop drives an ambient, time-based pulse independent of mouse movement or clicks. All modes (other than `off`) work alongside cursor and burst interaction — whichever produces the larger target scale for a given shape wins each frame.

| Mode | Behavior |
|------|----------|
| off | No ambient animation; shapes only react to cursor proximity and click bursts. |
| random | Each cell pulses on its own pseudo-random cycle (combining a slow sine wave and a faster one keyed to a per-cell seed), only crossing into visible territory above an internal threshold. Produces a sparse, organic flicker. |
| pulse | All cells pulse together in unison on a shared sine wave. |
| ripple | Concentric rings emanate from the grid's center outward on a repeating 3-second cycle. |
| left-to-right / right-to-left | A vertical band sweeps across the grid horizontally. |
| top-to-bottom / bottom-to-top | A horizontal band sweeps across the grid vertically. |
| line-right / line-left | Similar to the left-to-right/right-to-left sweep but with a narrower band, reading as a thinner moving line rather than a wide wave. |
| wave-horizontal | A continuous sine wave travels across the grid horizontally, with multiple peaks visible at once (period ≈ ⅓ of the grid width). |
| wave-vertical | Same as `wave-horizontal` but traveling vertically. |
| diagonal-tl-br / diagonal-br-tl | A diagonal band sweeps from top-left to bottom-right, or the reverse. |

### Full Component Source
```jsx
// JS-TW variant
import { useRef, useEffect } from "react";

const DEFAULT_COLORS = [
  "#7a5af8", "#d35af8", "#22c39b", "#f5b942", "#e0497a",
  "#3aa0e8", "#ffffff", "#ff6b6b", "#4ecdc4",
  { stops: ["#7a5af8", "#22c39b"] },
  { stops: ["#d35af8", "#3aa0e8"] },
  { stops: ["#f5b942", "#e0497a"] },
];

const TWO_PI = Math.PI * 2;

function smooth(x) {
  x = Math.max(0, Math.min(1, x));
  return x * x * (3 - 2 * x);
}

const RANDOM_MODE_DAMPING = 0.55;

const ShapesDots = ({
  width = "100%",
  height = "100%",
  className = "",
  children,

  cellSize = 40,
  influenceRadiusVmin = 30,
  attackTime = 0.5,
  releaseTime = 0.6,
  idleScale = 0.09,
  minPeakScale = 1,
  maxPeakScale = 3,

  burstSpeed = 1200,
  burstThickness = 180,

  backgroundColor = "#080808",
  shapes = ["circle", "triangle", "square"],
  colors = DEFAULT_COLORS,

  dpr = 2,
  opacity = 1,

  animationMode = "off",
  animationSpeed = 1,

  overlapGuard = 0.86,
  rotationVariance = 1,
}) => {
  const wrapRef = useRef(null);
  const canvasRef = useRef(null);

  const propsRef = useRef({});
  propsRef.current = {
    cellSize, influenceRadiusVmin, attackTime, releaseTime,
    idleScale, minPeakScale, maxPeakScale,
    burstSpeed, burstThickness,
    backgroundColor, shapes, colors,
    dpr, opacity,
    animationMode, animationSpeed, overlapGuard,
    rotationVariance,
  };

  const rebuildRef = useRef(false);
  const prevLayout = useRef({
    cellSize,
    shapes: JSON.stringify(shapes),
    colors: JSON.stringify(colors),
    minPeakScale,
    maxPeakScale,
    rotationVariance,
    overlapGuard,
  });

  useEffect(() => {
    const sig = {
      cellSize,
      shapes: JSON.stringify(shapes),
      colors: JSON.stringify(colors),
      minPeakScale,
      maxPeakScale,
      rotationVariance,
      overlapGuard,
    };
    if (
      sig.cellSize !== prevLayout.current.cellSize ||
      sig.shapes !== prevLayout.current.shapes ||
      sig.colors !== prevLayout.current.colors ||
      sig.minPeakScale !== prevLayout.current.minPeakScale ||
      sig.maxPeakScale !== prevLayout.current.maxPeakScale ||
      sig.rotationVariance !== prevLayout.current.rotationVariance ||
      sig.overlapGuard !== prevLayout.current.overlapGuard
    ) {
      prevLayout.current = sig;
      rebuildRef.current = true;
    }
  });

  useEffect(() => {
    const wrap = wrapRef.current;
    const canvas = canvasRef.current;
    const ctx = canvas.getContext("2d");

    let W = 0, H = 0;
    let cells = [];
    let cols = 0, rows = 0;
    let mouse = { x: -99999, y: -99999 };
    let bursts = [];
    let lastT = performance.now();
    let clock = 0;
    let rafId = null;

    function setSize() {
      const rect = wrap.getBoundingClientRect();
      W = rect.width;
      H = rect.height;
      const ratio = Math.min(window.devicePixelRatio || 1, propsRef.current.dpr);
      canvas.width = W * ratio;
      canvas.height = H * ratio;
      ctx.setTransform(ratio, 0, 0, ratio, 0, 0);
    }

    function pickColor() {
      const pool = propsRef.current.colors;
      return pool[Math.floor(Math.random() * pool.length)];
    }

    function pickShape() {
      const pool = propsRef.current.shapes;
      return pool[Math.floor(Math.random() * pool.length)];
    }

    function getBaseSize() {
      const { cellSize } = propsRef.current;
      return cellSize * 0.28;
    }

    function getHardCapScale() {
      const { cellSize, overlapGuard } = propsRef.current;
      const baseSize = getBaseSize();
      const sizeCap = cellSize * 0.5 * overlapGuard;
      return sizeCap / baseSize;
    }

    function makeGrid() {
      const { cellSize, idleScale, minPeakScale, maxPeakScale, rotationVariance } = propsRef.current;
      cols = Math.max(1, Math.floor(W / cellSize));
      rows = Math.max(1, Math.floor(H / cellSize));
      const padX = (W - (cols - 1) * cellSize) / 2;
      const padY = (H - (rows - 1) * cellSize) / 2;

      const hardCapScale = getHardCapScale();

      cells = [];
      for (let j = 0; j < rows; j++) {
        for (let i = 0; i < cols; i++) {
          const rawPeak = minPeakScale + Math.random() * (maxPeakScale - minPeakScale);
          cells.push({
            x: padX + i * cellSize,
            y: padY + j * cellSize,
            col: i,
            row: j,
            shape: pickShape(),
            color: pickColor(),
            peak: Math.min(rawPeak, hardCapScale),
            scale: idleScale,
            seed: Math.random() * 1000,
            phase: Math.random() * TWO_PI,
            rotation: Math.random() * TWO_PI * rotationVariance,
          });
        }
      }
    }

    function resolveColor(c, size) {
      if (typeof c === "string") return c;
      const grad = ctx.createLinearGradient(0, -size, 0, size);
      grad.addColorStop(0, c.stops[0]);
      grad.addColorStop(1, c.stops[1]);
      return grad;
    }

    function drawShape(shape, size, color) {
      ctx.save();
      ctx.fillStyle = resolveColor(color, size);
      if (shape === "circle") {
        ctx.beginPath();
        ctx.arc(0, 0, size, 0, TWO_PI);
        ctx.fill();
      } else if (shape === "square") {
        ctx.fillRect(-size, -size, size * 2, size * 2);
      } else if (shape === "triangle") {
        ctx.beginPath();
        ctx.moveTo(0, -size);
        ctx.lineTo(size * 0.87, size * 0.5);
        ctx.lineTo(-size * 0.87, size * 0.5);
        ctx.closePath();
        ctx.fill();
      }
      ctx.restore();
    }

    function animIntensity(c, t, mode) {
      if (mode === "off") return 0;
      switch (mode) {
        case "random": {
          const cycle = Math.sin(t * 0.6 + c.phase) * 0.5 + 0.5;
          const burst = Math.sin(t * 2.3 + c.seed) * 0.5 + 0.5;
          const combined = cycle * burst;
          const threshold = 0.82;
          const v = combined < threshold ? 0 : smooth((combined - threshold) / (1 - threshold));
          return v * RANDOM_MODE_DAMPING;
        }
        case "pulse": {
          const v = (Math.sin(t * TWO_PI * 0.5) + 1) / 2;
          const threshold = 0.75;
          return v > threshold ? smooth((v - threshold) / (1 - threshold)) : 0;
        }
        case "ripple": {
          const cx = (cols - 1) / 2, cy = (rows - 1) / 2;
          const dx = c.col - cx, dy = c.row - cy;
          const dist = Math.sqrt(dx * dx + dy * dy);
          const maxDist = Math.sqrt(cx * cx + cy * cy) || 1;
          const ringPos = dist / maxDist;
          const period = 3;
          const phase = (t % period) / period;
          const width = 0.12;
          const diff = Math.abs(ringPos - phase);
          return diff < width ? smooth(1 - diff / width) : 0;
        }
        case "left-to-right":
        case "right-to-left": {
          const pos = mode === "left-to-right"
            ? c.col / Math.max(cols - 1, 1)
            : 1 - c.col / Math.max(cols - 1, 1);
          const width = 0.12;
          const phase = (t % (1 + width)) - width / 2;
          const dist = Math.abs(pos - phase);
          return dist < width ? smooth(1 - dist / width) : 0;
        }
        case "top-to-bottom":
        case "bottom-to-top": {
          const pos = mode === "top-to-bottom"
            ? c.row / Math.max(rows - 1, 1)
            : 1 - c.row / Math.max(rows - 1, 1);
          const width = 0.12;
          const phase = (t % (1 + width)) - width / 2;
          const dist = Math.abs(pos - phase);
          return dist < width ? smooth(1 - dist / width) : 0;
        }
        case "line-right":
        case "line-left": {
          const pos = mode === "line-right"
            ? c.col / Math.max(cols - 1, 1)
            : 1 - c.col / Math.max(cols - 1, 1);
          const width = 0.06;
          const phase = (t % (1 + width)) - width / 2;
          const dist = Math.abs(pos - phase);
          return dist < width ? smooth(1 - dist / width) : 0;
        }
        case "wave-horizontal": {
          const phase = (c.col / Math.max(cols - 1, 1)) * Math.PI * 3 + t * TWO_PI;
          const v = (Math.sin(phase) + 1) / 2;
          return v > 0.72 ? smooth((v - 0.72) / 0.28) : 0;
        }
        case "wave-vertical": {
          const phase = (c.row / Math.max(rows - 1, 1)) * Math.PI * 3 + t * TWO_PI;
          const v = (Math.sin(phase) + 1) / 2;
          return v > 0.72 ? smooth((v - 0.72) / 0.28) : 0;
        }
        case "diagonal-tl-br":
        case "diagonal-br-tl": {
          let pos = ((c.col / Math.max(cols - 1, 1)) + (c.row / Math.max(rows - 1, 1))) / 2;
          if (mode === "diagonal-br-tl") pos = 1 - pos;
          const width = 0.1;
          const phase = (t % (1 + width)) - width / 2;
          const dist = Math.abs(pos - phase);
          return dist < width ? smooth(1 - dist / width) : 0;
        }
        default:
          return 0;
      }
    }

    function draw() {
      rafId = requestAnimationFrame(draw);

      if (rebuildRef.current) {
        rebuildRef.current = false;
        makeGrid();
      }

      const {
        influenceRadiusVmin, attackTime, releaseTime,
        idleScale, burstSpeed, burstThickness,
        backgroundColor, opacity, animationMode, animationSpeed, overlapGuard, cellSize,
      } = propsRef.current;

      const now = performance.now();
      const dt = Math.min((now - lastT) / 1000, 0.05);
      lastT = now;
      clock += dt * animationSpeed;

      ctx.clearRect(0, 0, W, H);
      if (backgroundColor && backgroundColor !== "transparent") {
        ctx.fillStyle = backgroundColor;
        ctx.fillRect(0, 0, W, H);
      }

      const vmin = Math.min(W, H);
      const influenceR = vmin * (influenceRadiusVmin / 100);

      for (let i = bursts.length - 1; i >= 0; i--) {
        bursts[i].radius += burstSpeed * dt;
        if (bursts[i].radius - burstThickness > Math.max(W, H) * 1.5) {
          bursts.splice(i, 1);
        }
      }

      ctx.globalAlpha = opacity;

      const baseSize = cellSize * 0.28;
      const sizeCap = cellSize * 0.5 * overlapGuard;

      for (const c of cells) {
        const dx = c.x - mouse.x, dy = c.y - mouse.y;
        const dist = Math.sqrt(dx * dx + dy * dy);

        let target = idleScale;
        if (dist < influenceR) {
          const f = 1 - dist / influenceR;
          target = Math.max(target, idleScale + (c.peak - idleScale) * f);
        }

        for (const b of bursts) {
          const bdx = c.x - b.x, bdy = c.y - b.y;
          const bdist = Math.sqrt(bdx * bdx + bdy * bdy);
          const ringDist = Math.abs(bdist - b.radius);
          if (ringDist < burstThickness / 2) {
            const f = 1 - ringDist / (burstThickness / 2);
            target = Math.max(target, idleScale + (c.peak - idleScale) * f);
          }
        }

        const animF = animIntensity(c, clock, animationMode);
        if (animF > 0) {
          target = Math.max(target, idleScale + (c.peak - idleScale) * animF);
        }

        const tau = target > c.scale ? attackTime : releaseTime;
        const alpha = 1 - Math.exp(-dt / Math.max(tau, 0.001));
        c.scale += (target - c.scale) * alpha;

        const size = Math.min(baseSize * c.scale, sizeCap);

        ctx.save();
        ctx.translate(c.x, c.y);
        ctx.rotate(c.rotation);
        drawShape(c.shape, size, c.color);
        ctx.restore();
      }
      ctx.globalAlpha = 1;
    }

    function onMove(e) {
      const rect = canvas.getBoundingClientRect();
      mouse.x = (e.clientX - rect.left) * (W / rect.width);
      mouse.y = (e.clientY - rect.top) * (H / rect.height);
    }
    function onLeave() {
      mouse.x = -99999;
      mouse.y = -99999;
    }
    function onClick(e) {
      const rect = canvas.getBoundingClientRect();
      const x = (e.clientX - rect.left) * (W / rect.width);
      const y = (e.clientY - rect.top) * (H / rect.height);
      bursts.push({ x, y, radius: 0 });
    }
    function onResize() {
      setSize();
      makeGrid();
    }

    setSize();
    makeGrid();
    rafId = requestAnimationFrame(draw);

    canvas.addEventListener("mousemove", onMove);
    canvas.addEventListener("mouseleave", onLeave);
    canvas.addEventListener("click", onClick);
    window.addEventListener("resize", onResize);

    return () => {
      cancelAnimationFrame(rafId);
      canvas.removeEventListener("mousemove", onMove);
      canvas.removeEventListener("mouseleave", onLeave);
      canvas.removeEventListener("click", onClick);
      window.removeEventListener("resize", onResize);
    };
  }, []);

  const classes = `relative overflow-hidden ${className}`;

  return (
    <div ref={wrapRef} className={classes} style={{ width, height }}>
      <canvas ref={canvasRef} className="block w-full h-full" style={{ cursor: "pointer" }} />
      {children}
    </div>
  );
};

export default ShapesDots;
```

### Integration Instructions
1. No external dependencies required — uses the native Canvas API only.
2. Copy the component source above into `./ShapesDots.jsx` in your project.
3. Import and render using the usage example above as a starting point.
4. The component sizes itself from its own `width`/`height` props (default `"100%"`), not from `absolute inset-0` against a parent — give the parent an explicit size if you want `100%` to resolve to something concrete:
   ```jsx
   <div style={{ width: '100%', height: '400px' }}>
     <ShapesDots />
   </div>
   ```
5. The grid listens for `window` resize events and recomputes both canvas size and grid layout (`setSize()` + `makeGrid()`) — no manual handling needed. Note this is a `window` resize listener, not a `ResizeObserver` on the container, so the grid won't auto-adjust if only the container (not the window) resizes.
6. Hovering the canvas grows nearby shapes toward their per-cell `peak` scale based on distance from the cursor (within `influenceRadiusVmin`). Clicking adds an expanding ring "burst" (`burstSpeed`, `burstThickness`) that grows shapes as it passes through them, independent of the cursor's current position.
7. `cellSize`, `shapes`, `colors`, `minPeakScale`, `maxPeakScale`, `rotationVariance`, and `overlapGuard` trigger a full grid rebuild (new random shape/color/peak/rotation assignments) on change — this is detected via a `JSON.stringify`-based signature comparison checked every render, not a dependency array, so passing a new array/object literal for `shapes` or `colors` on every render will rebuild every render even if the contents are identical only by reference, not by value (the stringify check avoids that specific footgun, but keep these props referentially stable or memoized where convenient anyway). All other props (`influenceRadiusVmin`, `attackTime`, `releaseTime`, `idleScale`, `burstSpeed`, `burstThickness`, `backgroundColor`, `dpr`, `opacity`, `animationMode`, `animationSpeed`) update live on the next animation frame via `propsRef` — no rebuild needed.
8. Each shape's `peak` scale is randomly chosen between `minPeakScale` and `maxPeakScale`, then hard-clamped so it can never exceed what `overlapGuard` allows for the current `cellSize` — this prevents neighboring shapes from visually overlapping even at maximum hover/click/animation influence.
9. Use `colors` entries of the form `{ stops: ["#colorA", "#colorB"] }` to render a shape with a vertical linear gradient fill instead of a solid color; mix solid hex strings and gradient objects freely in the same array.
10. Set `animationMode` to anything other than `"off"` to add an ambient pulse layered on top of cursor/click interaction — see the **Animation Modes** table above for what each value does. `animationSpeed` scales the internal clock driving these patterns independent of `attackTime`/`releaseTime`.
11. Use `children` to layer UI content over the canvas — note that unlike some other components in this set, the children here are **not** wrapped in a `relative z-10` div, so stacking order relies on normal DOM flow/`position` rules; add your own positioning if needed.
12. `dpr` caps device pixel ratio for canvas resolution exactly like in other Canvas-based components in this set — lower it (e.g. to `1`) if you need to trade visual sharpness for performance on dense grids.
