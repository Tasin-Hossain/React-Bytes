## Integrate the <Vortex /> component from React Bytes

You are helping integrate an open-source React component into an existing application.

### Component: Vortex
### Variant: JavaScript + Tailwind
### Dependencies: @react-three/fiber, three

---

### Usage Example
```jsx
import Vortex from './Vortex'

<Vortex
  speed={0.8}
  scale={2.0}
  colorA="#1B0F33"
  colorB="#FF5DA2"
  noiseIntensity={1.5}
  blend={0.5}
  opacity={1.0}
  bgColor="#000000"
  className=""
/>
```

### Props
| Prop | Type | Default | Description |
|------|------|---------|-------------|
| speed | number | 0.8 | Animation speed multiplier for the spiral/swirl movement. |
| scale | number | 2.0 | Zoom scale of the vortex pattern. Higher = more zoomed in. |
| colorA | string | "#1B0F33" | Primary color of the vortex spiral (hex). |
| colorB | string | "#FF5DA2" | Secondary/accent color — used for the swirl mix and center glow (hex). |
| noiseIntensity | number | 1.5 | Strength of the Silk-style grain noise applied over the pattern (0 = none). |
| blend | number | 0.5 | Controls the smoothstep alpha blend width between the background and the vortex colors. Lower = harder edge, higher = softer fade. |
| opacity | number | 1.0 | Overall opacity of the rendered canvas (0–1). |
| bgColor | string | "#000000" | Background color — applied both as a CSS fill on the wrapper div and as the GLSL blend target inside the shader (hex). |
| className | string | "" | Additional CSS classes applied to the outer wrapper div. |

### Full Component Source
```jsx
import { Canvas, useFrame, useThree } from '@react-three/fiber';
import { forwardRef, useRef, useMemo, useLayoutEffect } from 'react';
import { Color } from 'three';

const hexToNormalizedRGB = hex => {
  hex = hex.replace('#', '');
  return [
    parseInt(hex.slice(0, 2), 16) / 255,
    parseInt(hex.slice(2, 4), 16) / 255,
    parseInt(hex.slice(4, 6), 16) / 255
  ];
};

const vertexShader = `
varying vec2 vUv;
void main() {
  vUv = uv;
  gl_Position = projectionMatrix * modelViewMatrix * vec4(position, 1.0);
}
`;

const fragmentShader = `
varying vec2 vUv;
uniform float uTime;
uniform vec3  uColorA;
uniform vec3  uColorB;
uniform vec3  uBgColor;
uniform float uSpeed;
uniform float uScale;
uniform float uNoiseIntensity;
uniform float uBlend;
uniform float uOpacity;
uniform vec2  uResolution;

const float e = 2.71828182845904523536;

float noise(vec2 texCoord) {
  float G = e;
  vec2  r = (G * sin(G * texCoord));
  return fract(r.x * r.y * (1.0 + texCoord.x));
}

void main() {
  vec2 uv = (vUv - 0.5) * uScale;
  uv.x   *= uResolution.x / uResolution.y;
  float t  = uTime * uSpeed;

  float angle  = atan(uv.y, uv.x);
  float radius = length(uv);

  float spiral = sin(angle * 4.0 + radius * 9.0 - t * 2.2) * 0.5 + 0.5;
  float swirl  = sin(radius * 16.0 - t * 3.0 + angle * 2.0) * 0.5 + 0.5;

  vec3 col = mix(uColorA, uColorB, spiral);
  col *= 0.55 + 0.45 * swirl;

  float glow = exp(-radius * 2.2);
  col += uColorB * glow * 0.5;

  float vig = smoothstep(1.4, 0.3, radius);
  col *= vig;

  float rnd = noise(gl_FragCoord.xy);
  col -= rnd / 15.0 * uNoiseIntensity;

  float intensity = dot(col, vec3(0.299, 0.587, 0.114));
  float midPoint  = 0.20;
  float alpha     = smoothstep(midPoint - uBlend * 0.5, midPoint + uBlend * 0.5, intensity);
  col = mix(uBgColor, col, alpha);

  gl_FragColor = vec4(col, uOpacity);
}
`;

const VortexPlane = forwardRef(function VortexPlane({ uniforms }, ref) {
  const { viewport } = useThree();

  useLayoutEffect(() => {
    if (ref.current) {
      ref.current.scale.set(viewport.width, viewport.height, 1);
    }
  }, [ref, viewport]);

  useFrame((_, delta) => {
    ref.current.material.uniforms.uTime.value += delta;
  });

  return (
    <mesh ref={ref}>
      <planeGeometry args={[1, 1, 1, 1]} />
      <shaderMaterial
        uniforms={uniforms}
        vertexShader={vertexShader}
        fragmentShader={fragmentShader}
        transparent
      />
    </mesh>
  );
});

const Vortex = ({
  speed          = 0.8,
  scale          = 2.0,
  colorA         = '#1B0F33',
  colorB         = '#FF5DA2',
  noiseIntensity = 1.5,
  blend          = 0.5,
  opacity        = 1.0,
  bgColor        = '#000000',
  className      = '',
}) => {
  const meshRef = useRef();

  const uniforms = useMemo(
    () => ({
      uTime:           { value: 0 },
      uSpeed:          { value: speed },
      uScale:          { value: scale },
      uColorA:         { value: new Color(...hexToNormalizedRGB(colorA)) },
      uColorB:         { value: new Color(...hexToNormalizedRGB(colorB)) },
      uBgColor:        { value: new Color(...hexToNormalizedRGB(bgColor)) },
      uNoiseIntensity: { value: noiseIntensity },
      uBlend:          { value: blend },
      uOpacity:        { value: opacity },
      uResolution:     { value: [1, 1] },
    }),
    [speed, scale, colorA, colorB, bgColor, noiseIntensity, blend, opacity]
  );

  return (
    <div
      className={`absolute inset-0 -z-10 ${className}`}
      style={{ backgroundColor: bgColor }}
    >
      <Canvas
        dpr={[1, 2]}
        frameloop="always"
        className="w-full! h-full!"
        onCreated={({ gl }) => {
          gl.setSize(
            gl.domElement.parentElement.clientWidth,
            gl.domElement.parentElement.clientHeight
          );
        }}
      >
        <VortexPlane ref={meshRef} uniforms={uniforms} />
      </Canvas>
    </div>
  );
};

export default Vortex;
```

### Integration Instructions
1. Install required dependencies:
   ```bash
   npm install @react-three/fiber three
   ```
2. Copy the component source above into `./Vortex.jsx` in your project.
3. Import and render using the usage example above as a starting point.
4. The component positions itself as `absolute inset-0 -z-10` — it is designed to sit behind sibling content. Wrap it in a `relative` positioned container:
   ```jsx
   <div style={{ position: 'relative', width: '100%', height: '400px' }}>
     <Vortex />
     <div style={{ position: 'relative', zIndex: 1 }}>Your content here</div>
   </div>
   ```
5. The internal `<Canvas>` from `@react-three/fiber` fills the wrapper automatically. Initial canvas size is explicitly set via `onCreated` to match the parent element's dimensions.
6. All props are fed into a GLSL shader via `useMemo`-created uniforms. Changing any prop recreates the uniform object — this is intentional and safe. `uTime` is excluded from the memo dependencies and accumulates continuously via `useFrame`.
7. `bgColor` is applied in two places: as a CSS `backgroundColor` on the wrapper div (prevents flash before canvas paints) and as the `uBgColor` uniform inside the shader (used as the blend target in the smoothstep alpha calculation).
8. `blend` controls how sharply the vortex fades into the background. Use `blend={0.1}` for a hard-edged vortex cutout; use `blend={0.9}` for a wide, soft atmospheric glow.
9. `noiseIntensity={0}` disables grain entirely for a clean, smooth render. Values above `2.0` produce a heavy film-grain effect.
10. Increase `scale` (e.g. `4.0`) to zoom into the center of the vortex; decrease it (e.g. `1.0`) to see more of the spiral arms.
11. The vortex center glow is always driven by `colorB` — swap `colorA` and `colorB` to invert which color dominates the spiral vs the glow.
