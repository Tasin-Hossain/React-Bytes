### Props
| Prop | Type | Default | Description |
|------|------|---------|-------------|
| text | string | "Be ✦ Creative ✦ With ✦ React ✦ Bytes ✦" | The marquee text. Repeated automatically to fill the curve path. |
| font | object | see below | Font styling object applied to the SVG `<text>` element. Supports `fontFamily`, `fontWeight`, `fontSize` (number, in px), `lineHeight`, `letterSpacing`, `textAlign`. |
| color | string | "#FFFFFF" | Fill color of the text (any valid CSS color). |
| direction | string | "left" | Scroll direction. Options: `"left"` \| `"right"` |
| baseVelocity | number | 18 | Base scroll speed. Internally scaled: `(baseVelocity / 100) * 800`. Higher = faster. |
| curveAmount | number | -574 | Controls the arc of the SVG quadratic bezier path. Negative = curves upward, positive = curves downward. `0` = flat horizontal line. |
| gap | number | 1 | Extra gap between text repetitions. Internally calculated as `(gap + 1) * 10` pixels. |
| draggable | boolean | true | Enables pointer drag to scrub the marquee. Drag velocity carries over as inertia after release. |
| dragIntensity | number | 8 | Multiplier for drag delta. Higher = more scroll per pixel dragged. Internally scaled by `0.1`. |
| fade | boolean | true | When true, applies an SVG mask that fades the text in and out at the left and right edges. |
| fadePercent | number | 50 | Controls the fade region width. The text is fully opaque between `fadePercent%` and `(100 - fadePercent)%` of the SVG width. Lower = wider fade. |
| className | string | "" | Additional CSS classes applied to the outer wrapper div. |
| style | object | {} | Inline styles applied to the outer wrapper div. |

### font prop defaults
```js
{
  fontFamily: 'sans-serif',
  fontWeight: 400,
  fontSize: 64,
  lineHeight: '1.5em',
  letterSpacing: '1px',
  textAlign: 'left'
```

### Full Component Source
```jsx
'use client';

import { useRef, useEffect, useState, useMemo } from 'react';

const MAX_SPEED = 800;

export default function CurvedMarquee({
  text = 'Be ✦ Creative ✦ With ✦ React ✦ Bytes ✦',
  font = {
    fontFamily: 'sans-serif',
    fontWeight: 400,
    fontSize: 64,
    lineHeight: '1.5em',
    letterSpacing: '1px',
    textAlign: 'left'
  },
  color = '#FFFFFF',
  direction = 'left',
  baseVelocity = 18,
  curveAmount = -574,
  gap = 1,
  draggable = true,
  dragIntensity = 8,
  fade = true,
  fadePercent = 50,
  style,
  className = ''
}) {
  const measureRef = useRef(null);
  const tspansRef = useRef([]);
  const pathRef = useRef(null);
  const [pathLength, setPathLength] = useState(0);
  const [textWidth, setTextWidth] = useState(0);

  const staticId = useMemo(() => {
    const propsString = `${text}-${curveAmount}-${direction}-${baseVelocity}`;
    let hash = 0;
    for (let i = 0; i < propsString.length; i++) {
      const char = propsString.charCodeAt(i);
      hash = (hash << 5) - hash + char;
      hash = hash & hash;
    }
    return Math.abs(hash).toString(36);
  }, [text, curveAmount, direction, baseVelocity]);

  const pathId = `curve-${staticId}`;
  const fadeGradientId = `fadeGradient-${staticId}`;
  const fadeMaskId = `fadeMask-${staticId}`;
  const pathD = `M-100,400 Q720,${400 + curveAmount} 1540,400`;

  const isDragging = useRef(false);
  const dragVelocity = useRef(0);
  const effectiveVelocity = (baseVelocity / 100) * MAX_SPEED;
  const actualBaseVelocity = direction === 'left' ? -effectiveVelocity : effectiveVelocity;
  const dragFactor = dragIntensity * 0.1;

  const gapPx = (gap + 1) * 10;
  const processedText = useMemo(() => text.trim(), [text]);
  const spacing = textWidth + gapPx;

  useEffect(() => {
    if (measureRef.current) {
      setTextWidth(measureRef.current.getComputedTextLength());
    }
  }, [text, font, color, direction, baseVelocity, curveAmount, gap, draggable, dragIntensity, fade, fadePercent]);

  useEffect(() => {
    if (pathRef.current) {
      setPathLength(pathRef.current.getTotalLength());
    }
  }, [curveAmount]);

  const calculatedRepeats = spacing > 0 ? Math.ceil(pathLength / spacing) + 2 : 0;
  const ready = pathLength > 0 && spacing > 0;

  useEffect(() => {
    if (!ready) return;
    let raf = 0;
    let last = performance.now();
    const tick = now => {
      const delta = now - last;
      last = now;
      const spans = tspansRef.current;
      if (spans.length > 0) {
        const maxX = (spans.length - 1) * spacing;
        if (isDragging.current) {
          for (const tspan of spans) {
            if (!tspan) continue;
            let x = parseFloat(tspan.getAttribute('x') || '0');
            x += dragVelocity.current;
            if (x < -spacing) x = maxX;
            if (x > maxX) x = -spacing;
            tspan.setAttribute('x', x.toString());
          }
          dragVelocity.current *= 0.9;
          if (Math.abs(dragVelocity.current) < 0.01) dragVelocity.current = 0;
        } else {
          let moveBy = actualBaseVelocity * (delta / 1e3);
          moveBy += dragVelocity.current;
          if (Math.abs(dragVelocity.current) > 0.01) {
            dragVelocity.current *= 0.96;
          } else {
            dragVelocity.current = 0;
          }
          for (const tspan of spans) {
            if (!tspan) continue;
            let x = parseFloat(tspan.getAttribute('x') || '0');
            x += moveBy;
            if (x < -spacing) x = maxX;
            if (x > maxX) x = -spacing;
            tspan.setAttribute('x', x.toString());
          }
        }
      }
      raf = requestAnimationFrame(tick);
    };
    raf = requestAnimationFrame(tick);
    return () => cancelAnimationFrame(raf);
  }, [ready, spacing, actualBaseVelocity]);

  const lastPointerPosition = useRef({ x: 0, y: 0 });

  const handlePointerDown = e => {
    if (!draggable) return;
    e.currentTarget.setPointerCapture(e.pointerId);
    e.currentTarget.style.cursor = 'grabbing';
    isDragging.current = true;
    lastPointerPosition.current = { x: e.clientX, y: e.clientY };
    dragVelocity.current = 0;
  };
  const handlePointerMove = e => {
    if (!draggable) return;
    if (!isDragging.current) return;
    const currentPosition = { x: e.clientX, y: e.clientY };
    const deltaX = currentPosition.x - lastPointerPosition.current.x;
    dragVelocity.current = deltaX * dragFactor;
    lastPointerPosition.current = currentPosition;
  };
  const handlePointerUp = e => {
    if (!draggable) return;
    e.currentTarget.releasePointerCapture(e.pointerId);
    e.currentTarget.style.cursor = 'grab';
    isDragging.current = false;
  };

  const cursorStyle = draggable ? (isDragging.current ? 'grabbing' : 'grab') : 'default';
  const fadeStart = `${fadePercent}%`;
  const fadeEnd = `${100 - fadePercent}%`;

  return (
    <div
      className={`w-full h-full flex items-center justify-center ${className}`}
      style={{ visibility: ready ? 'visible' : 'hidden', ...style }}
    >
      <svg
        viewBox="0 0 1440 800"
        className="absolute top-1/2 left-1/2 -translate-x-1/2 -translate-y-1/2 select-none w-full block overflow-visible"
        style={{
          aspectRatio: '1440 / 800',
          fill: color,
          fontFamily: font.fontFamily,
          fontSize: font.fontSize,
          letterSpacing: font.letterSpacing,
          lineHeight: font.lineHeight
        }}
      >
        <text
          ref={measureRef}
          xmlSpace="preserve"
          className="invisible opacity-0 pointer-events-none"
          style={{ cursor: cursorStyle }}
        >
          {processedText}
        </text>
        <defs>
          <path ref={pathRef} id={pathId} d={pathD} fill="none" stroke="transparent" />
          {fade && (
            <>
              <linearGradient id={fadeGradientId} x1="0%" y1="0%" x2="100%" y2="0%">
                <stop offset="0%" stopColor="white" stopOpacity="0" />
                <stop offset={fadeStart} stopColor="white" stopOpacity="1" />
                <stop offset={fadeEnd} stopColor="white" stopOpacity="1" />
                <stop offset="100%" stopColor="white" stopOpacity="0" />
              </linearGradient>
              <mask id={fadeMaskId}>
                <rect width="100%" height="100%" fill={`url(#${fadeGradientId})`} />
              </mask>
            </>
          )}
        </defs>
        {ready && (
          <text
            fontWeight={font.fontWeight}
            xmlSpace="preserve"
            mask={fade ? `url(#${fadeMaskId})` : undefined}
            onPointerDown={handlePointerDown}
            onPointerMove={handlePointerMove}
            onPointerUp={handlePointerUp}
            onPointerCancel={handlePointerUp}
          >
            <textPath href={`#${pathId}`} xmlSpace="preserve">
              {Array.from({ length: calculatedRepeats }).map((_, i) => (
                <tspan
                  key={i}
                  x={i * spacing}
                  ref={el => {
                    if (el) tspansRef.current[i] = el;
                  }}
                >
                  {processedText}
                </tspan>
              ))}
            </textPath>
          </text>
        )}
      </svg>
    </div>
  );
}
```

### Integration Instructions
1. No external dependencies required — uses the native SVG API only.
2. Copy the component source above into `./CurvedMarquee.jsx` in your project.
3. The `'use client'` directive at the top is required for Next.js App Router. Remove it if you are using Vite, CRA, or any non-Next.js setup.
4. Import and render using the usage example above as a starting point.
5. The component is `w-full h-full` and the internal SVG is `absolute` positioned. Wrap it in a sized, `relative` container:
   ```jsx
   <div style={{ position: 'relative', width: '100%', height: '300px' }}>
     <CurvedMarquee />
   </div>
   ```
6. The component is invisible (`visibility: hidden`) until both `pathLength` and `textWidth` are measured from the DOM — this prevents a flash of un-positioned text on first render. Both are measured via `getTotalLength()` and `getComputedTextLength()` on the SVG elements after mount.
7. SVG `<defs>` IDs (`pathId`, `fadeGradientId`, `fadeMaskId`) are generated from a stable hash of `text`, `curveAmount`, `direction`, and `baseVelocity`. This allows multiple `<CurvedMarquee>` instances on the same page without ID collisions.
8. `curveAmount` controls the quadratic bezier control point: the path is `M-100,400 Q720,{400 + curveAmount} 1540,400`. Negative values arc upward, positive values arc downward. `curveAmount={0}` produces a flat horizontal marquee.
9. `baseVelocity` is not in pixels-per-second directly — it is scaled internally as `(baseVelocity / 100) * 800`. A value of `18` equals `144px/s`. Adjust proportionally; the cap is effectively `MAX_SPEED = 800px/s` at `baseVelocity={100}`.
10. Drag uses `setPointerCapture` on the SVG `<text>` element — pointer events continue to fire even if the cursor leaves the element during drag. `onPointerCancel` is handled the same as `onPointerUp` to cover touch cancellation.
11. During drag (`isDragging.current = true`), the auto-scroll loop is paused and only `dragVelocity` drives position. On release, `dragVelocity` decays at `0.9x` per frame during drag-only mode, then transitions to `0.96x` decay when blended back with auto-scroll.
12. The number of text repetitions is calculated dynamically as `Math.ceil(pathLength / spacing) + 2` — the `+2` ensures there are always enough copies to fill the visible arc plus wrap seamlessly without gaps.
13. `fadePercent={50}` (default) fades in over the first 50% and fades out over the last 50% — the text is fully opaque only at the exact center. Use `fadePercent={20}` for a subtle edge fade with most of the text visible.
14. `font.fontSize` must be a plain number (pixels) — do not pass a string like `"64px"`. It is applied directly to the SVG `style.fontSize` attribute.
